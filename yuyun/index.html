渔云是什么

渔云（CloudFisher）2024 年前后上线，采用 IEPL 专线与公网中转结合的方案，提供多端定制客户端，主打长期稳定使用。

线路 · 协议 · 解锁

覆盖港日台新美等主流地区；采用较新的 VLESS、Hysteria2，官方称提升跨境访问与流媒体体验，支持 Netflix/Disney+/ChatGPT。

  • IEPL 专线 + 中转
  • 协议：VLESS / Hysteria2
  • 支持多端定制客户端
  • 全套餐赠送 Emby 影视库，登录即看片
具体测速因时段而异，建议实测。

新手上手：4 步开用

客户端提示：本机场含 Hysteria2 节点，请使用 v2rayN / Hiddify / Karing 等支持该协议的客户端；Xray-core、老版本 Clash 无法连接，导入会报错。
  1. 注册并购买套餐。打开 渔云 官网注册账号，按需求选一档套餐下单。
  2. 复制订阅链接。在用户中心找到“一键订阅 / 复制订阅地址”，复制你的专属链接。
  3. 安装客户端。Windows 用 Clash Verge / v2rayN，安卓用 Hiddify / Karing，iOS 用 Shadowrocket。
  4. 导入并选节点。把订阅粘贴进客户端、更新节点，挑一个延迟低的节点开启即可。
