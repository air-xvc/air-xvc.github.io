红杏云是什么

红杏云（HongXing）是 2024 年中上线的机场，与专线站「糖果云」同源。经过升级，红杏云现已全线采用 IEPL 专线传输，把资源砸在流媒体解锁与大流量性价比上，并赠送 EMBY 私人影视库。

注意：市面上有同名旧站与「一枝红杏」等，均非同一运营实体；红杏云备用域名较多，购买前请认准官方永久地址，避免钓鱼假站。

线路与节点

红杏云现全线走 IEPL 专线传输，晚高峰更抗拥堵、延迟更低；国际节点覆盖香港、日本、台湾、新加坡、美国等常用地区，原生 IP 较多、解锁友好。

  • 全线 IEPL 专线传输，晚高峰稳、延迟低
  • 热门节点：香港 / 日本 / 台湾 / 新加坡 / 美国
  • 设备不限制，支持家庭成员共享
  • 带宽随档位递增（300 → 1000Mbps）
周期套餐赠送 EMBY 影视库；不限时买断套餐主打大流量、用完为止（不含 EMBY 赠送）。

协议与客户端

采用 Trojan 与 Hysteria2 协议，兼容主流客户端，导入订阅即可用。

  • 支持 Clash、Shadowrocket、Quantumult X、Stash、sing-box、Hiddify 等
  • Hysteria2 节点需用支持该协议的客户端
  • 支付方式：支付宝 / 微信 / USDT

解锁与影视库

核心卖点是全流媒体解锁 + 影视库：稳定解锁 Netflix、Disney+、YouTube Premium 与 ChatGPT / OpenAI，4K 秒播；周期套餐赠送 EMBY 私人影视库，装上 EMBY 客户端登录即可直接看片。

优点与缺点

优点

  • 全线 IEPL 专线传输，晚高峰稳、延迟低
  • 月付四档赠送 EMBY 影视库，追剧免找资源
  • 大流量还便宜：¥80/1200G，性价比高
  • 独有不限时大流量买断（3000G ¥388 / 6000G ¥688）
  • 设备不限、支持家庭共享，全流媒体解锁
  • 支付方式多（支付宝 / 微信 / USDT）

要注意

  • 2024 年新站，建议先月付验证晚高峰
  • 不限时套餐不含 EMBY 赠送
  • 备用域名多，注意认准官方地址防钓鱼
  • 6000G 不限时档限 20 台设备

新手上手：4 步开用

客户端提示：本机场含 Hysteria2 节点，请使用 v2rayN / Hiddify / Karing 等支持该协议的客户端；Xray-core、老版本 Clash 无法连接，导入会报错。
  1. 注册并购买套餐。打开 红杏云 官网注册账号，按需求选一档套餐下单。
  2. 复制订阅链接。在用户中心找到“一键订阅 / 复制订阅地址”，复制你的专属链接。
  3. 安装客户端。Windows 用 Clash Verge / v2rayN，安卓用 Hiddify / Karing，iOS 用 Shadowrocket。
  4. 导入并选节点。把订阅粘贴进客户端、更新节点，挑一个延迟低的节点开启即可。

常见问题

红杏云和糖果云是什么关系？
两者同源。现在红杏云本站也已全线升级 IEPL 专线；区别在定位：红杏云更主打「性价比 + 大流量」，月付大流量档更便宜，还独有不限时大流量买断套餐；糖果云则更聚焦纯月付的追剧影视体验。按需求二选一即可。
不限时套餐和月付套餐有什么区别？
月付套餐每月按订单日重置流量、并赠送 EMBY 影视库；不限时套餐（3000G / 6000G）是一次性买断、用完为止，主打大流量长期备用（不含 EMBY 赠送）。
新站会不会跑路？
红杏云 2024 年中上线，属于新站。稳妥做法是先买月付（约 ¥20/200G）验证晚高峰稳定性，别一上来大额年付，并认准官方永久域名，避免钓鱼假站。
