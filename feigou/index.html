飞狗是什么

飞狗是一家走低价走量路线的机场，套餐清一色用萌系狗狗命名（博美 / 泰迪 / 柯基 / 边牧），主打「便宜、大流量、不限设备」，适合日常刷剧、多设备共享等场景。

线路 · 协议 · 解锁

公网隧道中转为主，覆盖港台日美等常见地区；全节点升级到 HY2 / VLESS / AnyTLS 新协议（抗封更好，需用教程里最新版客户端），可解锁 Netflix、YouTube、ChatGPT 等常见需求。

  • 所有套餐不限制在线客户端，多设备随便连
  • 全节点新协议：HY2 / VLESS / AnyTLS
  • 购买日免费重置流量；仅限日常使用、禁止 BT
公网中转更亲民，但抗干扰弱于专线；历史上换过网址，注意保存官网最新地址。

优点与缺点

优点

  • 所有套餐不限制在线客户端，多设备 / 全家共享无压力
  • 大流量低价：柯基 ¥6/500G、边牧 ¥12/1T
  • 全节点 HY2/VLESS/AnyTLS 新协议，抗封更好
  • 萌系狗狗命名、套餐简单，购买日免费重置

要注意

  • 历史上曾出现域名失联/更换网址，稳定性需谨慎评估
  • 机场行业存在跑路风险，建议先用月付或小额套餐验证稳定性，别一上来就大额年付。

适合谁 / 谁要三思

适合入手

  • 想要大流量又便宜、且多设备一起连的用户
  • 喜欢萌系品牌、找低价口粮 / 备用线路的用户

建议三思

  • 对稳定性要求高、用于工作商务的用户

新手上手：4 步开用

客户端提示：本机场含 Hysteria2 节点，请使用 v2rayN / Hiddify / Karing 等支持该协议的客户端；Xray-core、老版本 Clash 无法连接，导入会报错。
  1. 注册并购买套餐。打开 飞狗 官网注册账号，按需求选一档套餐下单。
  2. 复制订阅链接。在用户中心找到“一键订阅 / 复制订阅地址”，复制你的专属链接。
  3. 安装客户端。Windows 用 Clash Verge / v2rayN，安卓用 Hiddify / Karing，iOS 用 Shadowrocket。
  4. 导入并选节点。把订阅粘贴进客户端、更新节点，挑一个延迟低的节点开启即可。
