守候是什么

守候（守候网络）是成立于 2020 年的老牌机场，主打 Shadowsocks，提供公网中转与 IPLC 专线两种线路，运营时间较长。

线路 · 协议 · 解锁

节点覆盖港台新美等 10+ 地区，入口较丰富，提供 0.2x 等低倍率节点降低流量消耗；SS 协议兼容性好，官方称解锁 Netflix、Disney+、ChatGPT。

  • 中转 + IPLC 专线可选
  • 含低倍率（如 0.2x）节点
  • 协议：Shadowsocks
  • 提供 Emby 私人影视库，登录即看，追剧免找片源
低倍率省流量，但专线选项通常倍率更高，选购前看清倍率规则。

新手上手：4 步开用

客户端提示：主流协议，Clash 系 / Shadowrocket 等常见客户端都可导入，粘贴订阅即可。
  1. 注册并购买套餐。打开 守候 官网注册账号，按需求选一档套餐下单。
  2. 复制订阅链接。在用户中心找到“一键订阅 / 复制订阅地址”，复制你的专属链接。
  3. 安装客户端。Windows 用 Clash Verge / v2rayN，安卓用 Hiddify / Karing，iOS 用 Shadowrocket。
  4. 导入并选节点。把订阅粘贴进客户端、更新节点，挑一个延迟低的节点开启即可。
