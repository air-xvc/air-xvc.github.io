一分机场是什么

一分机场成立于 2023 年前后，主打「极致低价」，采用亚马逊、甲骨文等大厂云服务器的直连线路（无专线），面向预算有限、需要大流量的日常用户。

线路 · 协议 · 解锁

三网直连、官方称带宽较高（无 IEPL/IPLC 专线）；采用 VLESS、Hysteria2，官方称解锁 Netflix、ChatGPT、bilibili。

  • 直连线路，无专线
  • 依托大厂云（亚马逊/甲骨文等）
  • 协议：VLESS / Hysteria2
直连高峰期稳定性通常弱于专线，实际体验以自行实测为准。

优点与缺点

优点

  • 低价大流量，日常性价比高
  • 套餐档位多、选择灵活
  • 注册流程简单

要注意

  • 直连无专线，晚高峰/跨境可能波动
  • 机场行业存在跑路风险，建议先用月付或小额套餐验证稳定性，别一上来就大额年付。

建议三思

新手上手：4 步开用

客户端提示：本机场含 Hysteria2 节点，请使用 v2rayN / Hiddify / Karing 等支持该协议的客户端；Xray-core、老版本 Clash 无法连接，导入会报错。
  1. 注册并购买套餐。打开 一分机场 官网注册账号，按需求选一档套餐下单。
  2. 复制订阅链接。在用户中心找到“一键订阅 / 复制订阅地址”，复制你的专属链接。
  3. 安装客户端。Windows 用 Clash Verge / v2rayN，安卓用 Hiddify / Karing，iOS 用 Shadowrocket。
  4. 导入并选节点。把订阅粘贴进客户端、更新节点，挑一个延迟低的节点开启即可。
