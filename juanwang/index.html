卷王机场是什么

卷王机场（对外也以 YGPY 出现）主打「把性价比卷到极致」：最突出的是所有套餐都不限制客户端数量，配合年付白菜价与订单日自动重置，适合多设备、全家共享或软路由长期挂着用。

线路 · 协议 · 解锁

公开信息显示覆盖美/港/台/新/日/英/加/德/法/荷/俄等 11 个国家地区，数量较多；支持 Hysteria/Trojan/VLESS/VMess，理论兼容 Clash Meta、sing-box、Shadowrocket；流媒体解锁数据较少，ChatGPT/TikTok 更易满足。

  • 客户端数量不限制（多设备 / 全家共享无压力）
  • 覆盖 11 个国家/地区，入口优化中转
  • 协议：Hysteria/Trojan/VLESS/VMess
晚高峰稳定性等深度数据较少，建议实测；官方禁止滥用分享，发现封号。

优点与缺点

优点

  • 全套餐客户端数量不限制，多设备 / 全家共享无压力
  • 年付白菜价（¥20/年起），订单日自动重置
  • 年付 / 月付 / 不限时买断三种计费，按需灵活
  • 覆盖 11 个国家/地区，协议支持主流客户端

要注意

  • 第三方独立评测与长期数据较少
  • 免费套餐是否限速/限期需看官网细则
  • 机场行业存在跑路风险，建议先用月付或小额套餐验证稳定性，别一上来就大额年付。

适合谁 / 谁要三思

适合入手

  • 家里多设备、想一份订阅全家共享的用户
  • 预算敏感、想年付长期低价挂机的用户

建议三思

  • 对晚高峰稳定性要求极高、要看详尽第三方测速的谨慎型用户

新手上手：4 步开用

客户端提示：Trojan 协议兼容性好，Clash Verge / v2rayN / Shadowrocket 等主流客户端均可直接导入使用。
  1. 注册并购买套餐。打开 卷王机场 官网注册账号，按需求选一档套餐下单。
  2. 复制订阅链接。在用户中心找到“一键订阅 / 复制订阅地址”，复制你的专属链接。
  3. 安装客户端。Windows 用 Clash Verge / v2rayN，安卓用 Hiddify / Karing，iOS 用 Shadowrocket。
  4. 导入并选节点。把订阅粘贴进客户端、更新节点，挑一个延迟低的节点开启即可。

常见问题

客户端数量真的不限制吗？
官网套餐页标注所有档位「客户端数量：不限制」，多设备同时在线不受台数限制；但禁止滥用分享，发现会封号。
怎么买更划算？
长期用选年付（¥20/年起、每月自动重置流量）最划算；想先验证稳定性可买 ¥15 起的不限时小额包，实测晚高峰再加量。
