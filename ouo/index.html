OuO 是什么

OuO 成立于 2023 年，是一家公网隧道中转机场，节点覆盖地区较多，另有家宽 IP 节点等细分选项。

线路 · 协议 · 解锁

覆盖港日台新美德英等主流与冷门地区，SS 在部分时段有低倍率优惠；兼容 Clash、Shadowrocket、Surge、QuantumultX，官方称解锁 Netflix/Disney+/YouTube/TikTok/ChatGPT。

新手上手：4 步开用

客户端提示：主流协议，Clash 系 / Shadowrocket 等常见客户端都可导入，粘贴订阅即可。
  1. 注册并购买套餐。打开 OuO 官网注册账号，按需求选一档套餐下单。
  2. 复制订阅链接。在用户中心找到“一键订阅 / 复制订阅地址”，复制你的专属链接。
  3. 安装客户端。Windows 用 Clash Verge / v2rayN，安卓用 Hiddify / Karing，iOS 用 Shadowrocket。
  4. 导入并选节点。把订阅粘贴进客户端、更新节点，挑一个延迟低的节点开启即可。
