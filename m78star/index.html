M78星云是什么

M78星云（英文常写 M78 Nebula，名字取自《奥特曼》里奥特曼的故乡「M78 星云」）是一家 2023 年 8 月上线的老牌中转机场，官网托管在 GitHub。它走 BGP 中转 + IEPL 专线路线，落地节点 40+、协议为 Shadowsocks（支持 UDP），运营两年多、口碑中规中矩。

最有辨识度的一点是『稀有冷门国家节点』：除了港、新、日、台、美等热门地区，它还上了南极、尼日利亚、埃及、阿根廷、乌克兰、丹麦、印尼、俄罗斯、泰国、越南、菲律宾等 20+ 冷门地区，且地区还在持续增加——这类冷门落地 IP 在多数机场里很难找到。

稀有节点与线路

M78星云为中转机场，带宽足、三网 BGP 入口，电信 / 联通 / 移动均有优化，峰值可达 2000Mbps，晚高峰播 4K 也较稳。中高级套餐叠加广港 IEPL 游戏专线与 Premium 线路，延迟更低。

  • 热门 + 稀有共 40+ 落地：港 / 新 / 日 / 台 / 美 + 南极 / 尼日利亚 / 埃及 / 阿根廷 / 乌克兰 / 丹麦 / 印尼 / 俄罗斯 等 20+ 冷门地区
  • 三网 BGP 入口，峰值 2000Mbps，晚高峰 4K 较稳
  • 中级套餐不含广港 IEPL 游戏专线 / Premium 线路；超值 300G 及以上档位才可用
『稀有节点』适合要冷门落地 IP 的用户（特定区域解锁 / 注册），但冷门节点的稳定性与速度通常不如主流地区；游戏专线仅高档套餐可用。

协议与客户端

M78星云协议为 Shadowsocks（支持 UDP），需使用本站客户端，覆盖苹果、安卓、Apple TV、OpenWrt、Linux、PC 全平台；把订阅导入对应客户端即可使用。

只认本站客户端 / 订阅，导入其它客户端可能异常；订阅格式以官网面板下发为准，找官网请认准官方 GitHub 渠道、当心同名山寨站。

追剧与福利是 M78星云的另一大重点：中级及以上套餐免费用本站 Emby 影视库，内置超 4 万部电影 / 剧集 / 纪录片、大部分 4K HDR，免费观看。此外年付 / 季付高级档还直送迪士尼或奈飞成品账号『车位』（发工单获取，用了送账号则不能再叠加优惠券）。

  • Emby 4 万部 4K HDR 影视库，中级以上套餐免费用（不过期包送 6 个月）

优点与缺点

优点

  • 站内独一份的『稀有冷门国家节点』：南极 / 尼日利亚 / 埃及 / 阿根廷 / 乌克兰 / 丹麦 等 20+ 冷门地区，冷门落地 IP 随便挑
  • 自带 Emby 4 万部 4K HDR 影视库，中级以上套餐免费用，追剧不用另养 Emby 号
  • 年付 / 季付高级档直送迪士尼 / 奈飞成品账号『车位』，等于机场 + 流媒体账号打包
  • 2023 年老牌、运营两年+，三网 BGP 入口峰值 2000Mbps，中高档含广港 IEPL 游戏专线 + Premium 线路
  • 全平台客户端（苹果 / 安卓 / Apple TV / OpenWrt / Linux / PC），另有 ¥99 不过期 300G 流量包做备用

要注意

  • 主力是 BGP 中转、非全程 IPLC/IEPL 专线，中级套餐还不含游戏专线 / Premium 线路，追求极致稳定要买中高档
  • 协议只有 Shadowsocks、且只能用本站客户端，玩法相对单一，无 Hysteria2 等新协议
  • 冷门稀有节点稳定性 / 速度通常不如主流地区，图新鲜可以、跑重要业务要先测
  • 明确暂无退款政策、送的成品账号不保证恢复；机场行业有跑路风险，先月付小额验证再考虑长约

适合谁 / 谁要三思

适合入手

  • 要南极 / 尼日利亚 / 埃及等冷门稀有落地 IP 做特定区域解锁 / 注册的用户
  • 想白嫖 Emby 4 万部 4K 影视库、顺带薅迪士尼 / 奈飞成品账号的追剧党
  • 接受 BGP 中转、日常刷剧上网 + 偶尔游戏专线的用户
  • 喜欢奥特曼 / 二次元情怀命名、想找个老牌口碑机场的用户

建议三思

  • 只认全程 IPLC/IEPL 专线、对晚高峰延迟极敏感的重度用户
  • 要 Hysteria2 / VLESS 等新协议、或要用自己客户端的玩家
  • 介意暂无退款、要长期第三方强口碑背书的保守用户

常见问题

M78星云的『稀有节点』有什么用？
它上了南极、尼日利亚、埃及、阿根廷、乌克兰、丹麦、印尼、俄罗斯等 20+ 冷门地区落地，这类 IP 在多数机场很难找到，适合做特定区域的流媒体 / 服务解锁、注册或多区养号。但冷门节点稳定性和速度通常不如港 / 日 / 美等主流地区，跑重要业务前建议先小额测。
年付送的迪士尼 / 奈飞『车位』是什么？怎么领？
『车位』指赠送一个对应流媒体的成品账号：高级 500G / 顶级 1000G 年付送迪士尼、高级 650G 年付送奈飞，购买后发工单领取。注意——用了送的账号就不能再叠加优惠券；奈飞车位禁止登录电视端；如遇平台政策变动导致账号不可用，机场会给其它补偿但不保证恢复原账号。
哪些套餐能用 Emby 和游戏专线？
Emby 影视库：中级 150G 及以上周期套餐免费用，¥99 不过期包送 6 个月；广港 IEPL 游戏专线 + Premium 线路：中级套餐不含，需买超值 300G 及以上档位才能用。拿不准先月付中级或买不过期包试水，满意再升档。
M78星云会不会跑路？
它 2023 年 8 月上线、运营两年多，属于老牌中转机场，目前公开没有跑路记录，第三方评测口碑中规中矩。但任何机场都有跑路风险、且本站明确暂无退款，稳妥做法仍是先月付小额验证三网晚高峰，别一上来大额年付，并再备一两家机场容灾。
