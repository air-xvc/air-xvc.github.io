自由猫是什么

自由猫（FreeCat）成立于 2020 年前后，是运营较久的中转机场，核心卖点是 MPTCP（多路径 TCP）——同时聚合多条线路提升稳定性，而非依赖单一路径。

线路 · 协议 · 解锁

据评测拥有 100+ 节点、覆盖 20+ 国家地区，MPTCP 在单路径机场易堵掉速时能自动切换维持稳定；支持 SS/Trojan/VLESS，解锁 Netflix、Disney+、ChatGPT。

  • 100+ 节点，20+ 地区
  • MPTCP 多路径，抗晚高峰拥堵
  • 协议：SS / Trojan / VLESS

新手上手：4 步开用

客户端提示：Trojan 协议兼容性好，Clash Verge / v2rayN / Shadowrocket 等主流客户端均可直接导入使用。
  1. 注册并购买套餐。打开 自由猫 官网注册账号，按需求选一档套餐下单。
  2. 复制订阅链接。在用户中心找到“一键订阅 / 复制订阅地址”，复制你的专属链接。
  3. 安装客户端。Windows 用 Clash Verge / v2rayN，安卓用 Hiddify / Karing，iOS 用 Shadowrocket。
  4. 导入并选节点。把订阅粘贴进客户端、更新节点，挑一个延迟低的节点开启即可。
